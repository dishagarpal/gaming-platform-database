# Game Database Management System (C)

A C-based management system  to manage players, games, and gameplay data using structured programming and file handling.

## Features

* Add and store new players
* Update player game statistics
* Track games played by users
* Identify active and inactive players
* Analyze most active games
* Display top 5 games by total play time
* View current players in a game (sorted by score)
* Remove inactive guest users

## Skills and Concepts Used

* C Programming
* Data Structures (arrays, structures)
* File Handling (read, write, append)
* Searching and Sorting


## Tech Stack

* Language: C
* Tools: GCC Compiler, VS Code

## Files Generated

* `players.txt` – player records
* `games.txt` – game details
* `gameplay.txt` – gameplay history

## How to Run

```bash
gcc code.c -o game
./game
```

This project demonstrates real-world data management using core DSA concepts in C.
